# 带边纹的通用卡牌模板预览 (Card Templates with Edge Patterns)

为了进一步增强卡牌类型的可辨识度，我们在边框内侧引入了特定的**边缘条纹图案**。这是为了确保即便在不关注颜色的情况下，玩家也能通过图案纹理识别卡牌类型。

## 2. 图案设计说明 (中英对照参考)
由于图片生成工具基于通用引擎，图中的标签为英文示例，其对应的**五行卡棋中文标准**如下：

- **左侧：主动卡 (Active)**
  - **色调**: 幽蓝色
  - **条纹**: **菱形连珠纹**。由连续的几何菱形组成，象征博弈中的严密逻辑与环环相扣的布局。
- **中间：反制卡 (Counter)**
  - **色调**: 烈红色
  - **条纹**: **锯齿雷电纹**。尖锐的折线纹路，象征瞬间的反应、阻拦与反击的锋芒。
- **右侧：即时卡 (Instant)**
  - **色调**: 辉煌金
  - **条纹**: **圆滑云水纹**。流动的曲线纹路，象征气的顺滑流转与即刻生效的自然感。

## 3. 标准化应用
这套模板是全自动化的前端还原基础。在 `game.html` 的 CSS 实现中，我们将通过 `border-image` 或 `svg-pattern` 来精确还原这些边缘条纹，确保每张卡牌都符合这一视觉规范。
